# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in TapPass, please report it responsibly.

**Email:** [security@tappass.ai](mailto:security@tappass.ai)

We will acknowledge your report within 48 hours and provide a detailed response within 5 business days.

Please do not open public issues for security vulnerabilities.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.5.x   | Yes       |
| < 0.5   | No        |
